## Start meteor wallet

Start an `gele` node and and the app using meteor and open http://localhost:3000 in your browser:

    $ gele --rpccorsdomain "http://localhost:3000" --rpc        
Optional add `--unlock <your account>`

Starting the wall dapp using [Meteor](http://meteor.com/install)

```
sudo apt-get install curl
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo npm install npm -g
sudo apt install -y node-gyp
curl https://install.meteor.com/ | sh
```

    $ cd meteor-dapp-wallet/app
    $ meteor --port<default 3000>

Open a browser window and access the URL for your wallet:          
http://localhost:3000
